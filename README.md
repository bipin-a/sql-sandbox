# SQL Sandbox

Practice SQL in the browser with curated exercises, schema imports, generated sample data, and a DuckDB-powered query runner.

## What it does

- Runs SQL locally with DuckDB-Wasm
- Ships with curated interview-style exercises
- Supports schema-only imports and generated sample data
- Lets you reveal solutions and check answers on seeded exercises

## Local development

```bash
npm install
npm run dev
```

## Tests

```bash
npm test -- --run
npm run test:e2e
```
